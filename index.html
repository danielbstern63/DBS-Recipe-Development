Connect to Google Sheets
Each submission will append a new row to your spreadsheet. One-time setup.
STEP 1 — Create a new Google Sheet. Name the first tab Recipes.
STEP 2 — Go to Extensions → Apps Script. Delete any existing code and paste the script, then click Deploy → New Deployment → Web App. Set Execute as: Me and Who has access: Anyone. Copy the Web App URL.
STEP 3 — Paste that URL into the field below and click Connect.
Privacy note: Your URL is stored only in this browser. No data passes through any third-party server.
