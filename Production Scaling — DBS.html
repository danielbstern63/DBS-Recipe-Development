Connect to Recipe Sheet
This app reads recipes marked "Approved for Production" from your Google Sheet. One-time setup — uses a separate read-only endpoint.
STEP 1 — Open your existing Recipe Google Sheet. Go to Extensions → Apps Script.
STEP 2 — Create a new script file (File → New → Script file) named Production. Paste the read script below into it.
STEP 3 — Deploy it as a new Web App: Deploy → New Deployment → Web App. Set Execute as: Me, Who has access: Anyone. Copy the URL.
STEP 4 — Also update your existing doPost script to include a recipeId column — see "Recipe ID Setup" below.
